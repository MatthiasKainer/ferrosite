## Motivation

LLMs are good at generating code, but they struggle with languages
that have large, inconsistent syntax surfaces. AXON explores what a
language looks like when it's designed *for* LLM generation first,
and human readability second.

## Design Principles

- S-expression syntax (fully parenthesised, unambiguous)
- Explicit types everywhere (no inference for LLMs)
- No operator overloading, no implicit conversions
- Compiled to native via LLVM for performance
- Small, orthogonal standard library

## Status

Lexer and parser complete. Type checker ~60% done. 
LLVM code generation started. Not yet self-hosting.
